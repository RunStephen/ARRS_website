Longest Running Championships

This section is an attempt to list all national, regional, and world championship series that have been contested 50 or more times. It is based on the best information available to ADR but is undoubtably missing many championships.

Race dates and male/female champions/times for each series listed are sought. When a sufficiently informative set of champions has been collected, that series is added to the website. You can view the dates/winners for each championship so profiled by clicking on the name of the championship in the following list. If the name of the championship is NOT underlined, then it has not been profiled yet. The last four columns are the number of dates, champions, times, and venues missing (profiled series only). Outdoor track championships are counted for 3000m/2 mile, 5000m/3 mile, and 10,000m/6 mile/5 mile/4 mile as (three) distinct series.
