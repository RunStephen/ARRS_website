Longest Running Marathons

This section attempts to list all marathons world-wide that have been held 20 or more times (runnings). It is based on the best information available to ADR but is undoubtably missing some marathons and may not have credited all runnings to the marathons listed. In some cases, it is difficult to trace a "marathon" through changes in organizing groups, venues, and other glitches in the series. In such cases, an effort has been made to maintain some measure of constancy in the series.

Race dates and male/female winners/times for each marathon listed are sought. When a sufficiently informative set of winners has been collected, a race history for that marathon is added to the website. You can view the dates/winners for each marathon so profiled by clicking on the name of the marathon in the following list.
